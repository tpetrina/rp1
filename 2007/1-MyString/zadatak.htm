1. zadaća

Opis zadatka

Napišite strukturu MyString koja predstavlja string duljine manje od 100. Deklaraciju strukture spremite u datoteku MyString.h, a implementaciju u datoteku MyString.cpp.
Uputa: vaša struktura treba kao element imati najobičniji C-ovski string duljine 100, poanta je da ispravno napišete funkcije članice.

Konstruktori:

MyString()kreira prazan string
MyString(char *str)kreira string koji je jednak C-ovskom stringu str
MyString(int n, char ch)kreira string koji se sastoji od n znakova ch (ako je n>99, onda kreira prazan string)

Funkcije:

int size()vraća duljinu stringa
char character(int n)vraća n-ti znak stringa (tj. onaj na indexu n; pretpostavite da je n uvijek manji od duljine stringa)
void append(MyString p)stringu koji poziva ovu funkciju se na kraj zalijepi string p
void ispisi()ispisuje string (bez ikakvih dodatnih poruka ili prelaska u novi red)
void copy(char *str)kopira C-ovski string str u string koji je pozvao ovu funkciju
void erase(int n, int q)briše q znakova stringa, počevši od znaka sa indexom n (pretpostavite da se sve odvija unutar granica stringa)
int isEqual(MyString p)provjerava da li je string jednak stringu p; vraća 1 ako je, 0 ako nije


Oprez:
  • struktura i funkcije MORAJU se zvati točno kako gore piše. Pazite na velika i mala slova!
  • vi trebate poslati samo datoteke MyString.h i MyString.cpp
  • u njima se ne smije nalaziti funkcija main()
  • također, niti jedna od funkcija (osim copy) koju napišete ne smije ništa učitavati niti ispisivati na ekran ili datoteku!!!

Ispravnost vašeg programa će biti provjerena tako da ćemo mi napisati razne main.cpp programe koji će kreirati varijable tipa MyString i pozivati funkcije koje ste vi napravili. Ako se vaše datoteke ne budu uspješno "linkale" sa našim klijentskim programom, vaše rješenje će se smatrati neispravnim.

Naravno, da provjerite da li vaš program radi prije nego ga pošaljete, također ga trebate istestirati pomoću nekakvog klijentskog programa. Slijedi primjer jednog takvog (ako vaša struktura MyString radi sa donjim main.cpp, to nije jamstvo da će raditi i nekim drugim, na primjer prilikom ocjenjivanja!)

Napomene:
  • morate odabrati svaku od gornjih datoteka i tek onda kliknuti na "Pošalji rješenja!"
  • svaka od datoteka smije imati najviše 10kB
  • nakon uspješnog slanja dobit ćete odgovarajuću poruku; u protivnom će se ponovno prikazati ova stranica
  • svoja rješenja smijete (uspješno) poslati još maksimalno 3 puta! Neuspješna slanja se ne broje.
  • ako rješenje šaljete više puta, prethodno poslana se gube!
